## 类与对象

### 类
组成: 成员变量(age)和成员方法print()。
定义: 对象的模板或蓝图。

### 对象
实例化: 使用new关键字。
引用: 对象变量存储的是内存地址。

### 构造方法
作用: 在创建对象时进行初始化操作。
特征: 方法名与类名完全相同，并且没有返回值类型。
可以有无参构造，也可以有带参数的构造方法。

### this 关键字
含义: 代表当前对象本身。
作用: 主要用于区分成员变量和局部变量。如构造方法中 `this.name = name;`。

## 封装结构

### 访问修饰符
- private
仅在本类内部可以访问。将属性设为private后，通过公开的set方法修改值，get方法获取值，以此保护数据安全。

- default
仅在同一个包内可见。

- protected
对子类和同一个包可见。

- public
对所有类开放无访问限制。

### 特殊修饰符
- static
静态变量：被该类所有对象共享。可以通过类名调用或对象调用。
静态方法：被类直接调用。

- final
修饰变量：变为常量一旦赋值不可修改。
修饰方法：表示该方法是最终版本，不能被子类重写。
修饰类：表示该类不能被继承。

## 继承结构

### 继承机制
使用extends建立子类与父类的关系。
Java是单继承，一个子类只能有一个直接父类。

### 成员变量的隐藏
当子类定义了和父类同名的成员变量时，父类的变量在子类中被隐藏，不再直接可见。

### super关键字
访问父类成员：使用 `super.变量名` 可以访问被隐藏的父类变量。
调用父类构造器：使用 `super(参数)` 调用父类的构造方法，且必须放在子类构造方法的第一行。

## 多态结构

### 定义
对象的多种形态。

### 使用
创建父类子类，在子类中使用@Override对方法进行重写。
创建对象：
```java
Fu a = new Zi();
```
调用方法：
```java
register(a);
```

### 调用的特点
调用变量: 编译看左边，运行看左边。
编译时看父类中有没有这个变量，有则编译成功。运行时输出父类中变量的赋值。

调用方法: 编译看左边，运行看右边。
编译时看父类中有没有这个方法，有则编译成功。运行时使用子类中的方法。

## 抽象与接口结构

### 抽象类与抽象方法

- 抽象类
特点：抽象类中不一定有抽象方法；不能创建对象；子类要么@Override重写抽象类中的所有抽象方法，要么也是抽象类。
方法声明：public abstract class ClassName {}
```java
public abstract class Animal {
}
```

- 抽象方法
有抽象方法的一定是抽象类。
方法声明：public abstract 返回值类型 方法名(参数列表);
```java
public abstract void eat();
```

### 接口

- 定义
不能创建对象。
方法声明：public interface InterfaceName {}
```java
public interface Playable {
}
```

- 使用
可以多实现，也可以继承后实现。
```java
public class Dog implements Playable, Runnable {
}

public class Cat extends Animal implements Playable {
}
```

- 成员的特点
成员变量：只能是常量，默认修饰词public static final。
构造方法：没有。
成员方法：可以定义抽象方法，有方法体的方法，私有方法。

## 内部类

### 定义
在一个类A的内部再定义一个类B。A称为外部类，B称为内部类，其他类统称为外部其他类。

### 使用场景
1. 内部类表示的事物是外部类的一部分。
2. 内部类单独出现没有意义。

### 访问特点
内部类可以访问外部类的成员，包括private。
外部类要访问内部类的成员，必须创建对象。

### 分类
成员内部类、静态内部类、局部内部类、匿名内部类。

### 匿名内部类
作用：定义类的同时直接实例化。整体是一个对象，可以在后面直接调用已经重写的方法，也可以作为参数使用。

结构与implements写法：
```java
Thread thread = new Thread(new Runnable() {
    @Override
    public void run() {
        System.out.println("Running...");
    }
});
```

结构与extends写法：
```java
Animal myDog = new Animal("旺财") {
    @Override
    public void speak() {
        System.out.println(getName() + " 汪汪汪！");
    }
};
```
