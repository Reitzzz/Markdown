# 初始化参数

初始化参数类似于项目启动配置所需的一些值（例如数据库连接信息或其他配置项），可以通过初始化参数传递给 Servlet。

## 1. 局部初始化参数

可以为单个 Servlet 添加初始化参数，数据以键值对形式保存：

```java
@WebServlet(value = "/login", initParams = {
        @WebInitParam(name = "test", value = "我是一个默认的初始化参数")
})
```

可以直接通过 Servlet 的 `getInitParameter` 方法获取该局部参数：

```java
System.out.println(getInitParameter("test"));
```

## 2. 全局初始化参数

如果配置项需要被整个 Web 应用程序共享，可以在 `web.xml` 中定义全局初始化参数：

```xml
<context-param>
    <param-name>lbwnb</param-name>
    <param-value>我是全局初始化参数</param-value>
</context-param>
```

读取全局初始化参数时，必须通过 `ServletContext` 对象来获取：

```java
ServletContext context = getServletContext();
System.out.println(context.getInitParameter("lbwnb"));
```
