## 1. 创建字符串

### 1.1 直接赋值
```java
String s1 = "abc";
```

### 1.2 new 关键字
```java
String s2 = new String();
String s3 = new String("abc");
```

### 1.3 字符数组
```java
char[] chs = {'a', 'b', 'c'};
String s4 = new String(chs);
```

### 1.4 字节数组
```java
byte[] bytes = {97, 98, 99};
String s5 = new String(bytes);
```

## 2. 字符串比较

### 2.1 == 与 equals
- `==`：基本类型比值，引用类型比地址。
- `equals()`：比较内容是否相等。

```java
boolean isEqual = s1.equals(s2);
```

### 2.2 equalsIgnoreCase
忽略大小写比较内容。
```java
boolean isEqualIgnoreCase = s1.equalsIgnoreCase(s2);
```

## 3. StringBuilder 与 StringBuffer

### 3.1 StringBuilder 构造方法
```java
StringBuilder sb1 = new StringBuilder();        // 空
StringBuilder sb2 = new StringBuilder("abc");   // 带初始值
```

### 3.2 StringBuilder 常用方法
| 方法 | 说明 |
|------|------|
| `append(Object)` | 追加数据，返回本身（链式调用） |
| `reverse()` | 反转内容 |
| `length()` | 返回长度 |
| `toString()` | 转换为 String |
| `compareTo(StringBuilder)` | 比较大小 |
| `insert(int, char)` | 插入字符 |
| `setCharAt(int, char)` | 修改字符 |

### 3.3 StringBuffer
方法与 `StringBuilder` 相同，但**线程安全**（方法用 `synchronized` 修饰）。

## 4. 遍历与操作字符串
| 方法 | 说明 |
|------|------|
| `charAt(int)` | 返回指定索引字符 |
| `length()` | 返回字符串长度 |
| `substring(int, int)` | 截取子串（左闭右开） |
| `replace(CharSequence, CharSequence)` | 替换字符串 |

## 5. 正则表达式
使用 `matches(String regex)` 进行匹配。

常用限定符：
| 字符 | 说明 |
|------|------|
| `*` | 零次或多次 |
| `+` | 一次或多次 |
| `?` | 零次或一次 |
| `{n}` | 恰好 n 次 |
| `{n,}` | 至少 n 次 |
| `{n,m}` | n 到 m 次 |

常用字符类：
| 字符 | 说明 |
|------|------|
| `[abc]` | a、b 或 c |
| `[^abc]` | 除 a、b、c 外 |
| `[a-zA-Z]` | 所有字母 |
| `\w` | 字母、数字、下划线 |
| `.` | 任意字符（除换行） |

## 6. Java 11+ 字符串增强
| 方法 | 说明 |
|------|------|
| `strip()` | 去除前后空格 |
| `stripLeading()` | 去除前导空格 |
| `stripTrailing()` | 去除尾部空格 |
| `isBlank()` | 是否为空或全空格 |
| `repeat(int)` | 重复字符串 n 次 |

```java
String s = "  Hello  ";
System.out.println(s.strip());          // "Hello"
System.out.println(s.isBlank());        // false
System.out.println("ABC".repeat(3));    // "ABCABCABC"
```

## 7. 文本块（Java 15+）
使用 `"""` 定义多行字符串，无需转义。

```java
String html = """
              <html>
                  <body>Hello</body>
              </html>
              """;
```
