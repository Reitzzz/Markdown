## 创建

### 1. 直接赋值获取一个字符串对象
```java
String s1 = "abc";
```

### 2. 使用new的方式来获取一个字符串对象
```java
String s2 = new String();
```

### 3. 传递一个字符串根据传递的字符串内容在创建一个新的字符串对象
```java
String s3 = new String("abc");
```

### 4. 传递一个字符数组，根据字符数组的内容，再创建一个新的字符串对象
```java
char[] chs = {'a', 'b', 'c'};
String s4 = new String(chs);
```

### 5. 传递一个字节数组，根据字节数组的内容，再创建一个新的字符串对象
```java
byte[] bytes = {97, 98, 99};
String s5 = new String(bytes);
```

## 比较
==号比较的内容：基本数据类型比较的是数据值；引用数据类型比较的是地址值。

### 方法

- 判断字符串内容是否相等
方法声明：public boolean equals(Object anObject)
```java
boolean isEqual = s1.equals(s2);
```

- 忽略大小写判断字符串内容是否相等
方法声明：public boolean equalsIgnoreCase(String anotherString)
```java
boolean isEqualIgnoreCase = s1.equalsIgnoreCase(s2);
```

## StringBuilder

### 用途
字符串拼接，字符串反转。支持链式编程。

### 构造方法

- 创建一个空白可变字符串
方法声明：public StringBuilder()
```java
StringBuilder sb1 = new StringBuilder();
```

- 根据传入字符串创建
方法声明：public StringBuilder(String str)
```java
StringBuilder sb2 = new StringBuilder("abc");
```

### 常用方法

- 添加数据，并返回对象本身
方法声明：public StringBuilder append(Object obj)
```java
sb1.append("def");
```

- 反转容器中的内容
方法声明：public StringBuilder reverse()
```java
sb1.reverse();
```

- 返回长度
方法声明：public int length()
```java
int len = sb1.length();
```

- 把StringBuilder转换String
方法声明：public String toString()
```java
String str = sb1.toString();
```

- 比较大小 (>0 表示 str1 > str2)
方法声明：public int compareTo(StringBuilder another)
```java
int result = sb1.compareTo(sb2);
```

- 插入字符
方法声明：public StringBuilder insert(int offset, char c)
```java
sb1.insert(1, 'z');
```

- 修改字符
方法声明：public void setCharAt(int index, char ch)
```java
sb1.setCharAt(0, 'x');
```

## StringBuffer
方法同StringBuilder，但StringBuffer是线程安全的，StringBuilder是线程不安全的。

## 遍历与操作字符串

- 根据索引返回字符
方法声明：public char charAt(int index)
```java
char c = s1.charAt(0);
```

- 返回字符串长度
方法声明：public int length()
```java
int len = s1.length();
```

- 截取字符串 (包左不包右)
方法声明：public String substring(int beginIndex, int endIndex)
```java
String sub = s1.substring(0, 2);
```

- 替换字符串
方法声明：public String replace(CharSequence target, CharSequence replacement)
```java
String replacedStr = s1.replace("a", "x");
```
