# IntelliJ IDEA Java开发核心快捷键笔记

## 一、 代码生成与智能提示
这类快捷键是IDEA的灵魂，能帮你省去大量手动编写模板代码的时间，特别是在处理实体类和配置类时。

* **`Alt + Enter`**：万能快捷键。用于显示意图动作和快速修复。遇到报错时按下它，IDEA会自动提供导包、添加异常处理或转换代码结构的建议。
* **`Alt + Insert`**：代码生成器。在类内部按下，可以一键生成Getter/Setter、构造函数、`hashCode`、`equals`和`toString`方法。
* **`Ctrl + Alt + V`**：自动提取变量。写完右侧的表达式后按下，IDEA会自动在左侧推导出类型并声明变量，极大地提高输入速度。
    * *进阶技巧（后缀补全）*：可以直接在语句最末尾敲上 `.var`，然后按回车或 Tab 键。效果与 `Ctrl + Alt + V` 完全一样，且打字时手指不需要离开主键盘区。
* **`Ctrl + O`**：重写父类方法。
* **`Ctrl + I`**：实现接口方法。当类实现了一个接口，按下此键可以快速批量生成所有未实现的方法签名。
* **`Ctrl + Shift + Enter`**：自动补全当前语句。会自动加上行尾的分号，或者补全`if`、`for`语句的大括号，并跳转到适合输入的位置。

## 二、 导航与代码阅读
在复杂的项目结构中，这组快捷键可以让你在多个文件和方法之间自由穿梭。

* **`双击 Shift`**：随处搜索 (Search Everywhere)。可以查找类、文件、操作或快捷键，是最高频的搜索入口。
* **`Ctrl + N`**：查找类。专门用于快速打开Java类文件。
* **`Ctrl + Shift + N`**：查找文件。不仅限于Java类，可以用来找配置文件（如XML或属性文件）或前端资源。
* **`Ctrl + B` (或 `Ctrl + 鼠标左键`)**：跳转到声明或用法。点按变量或方法名，可以直接跳转到它定义的地方；如果在定义处按下，则会列出它被调用的所有位置。
* **`Alt + F7`**：查找所有使用情况。在底部面板详细列出某个类、方法或变量在整个项目中的所有引用。

## 三、 代码重构与整理
在项目迭代或复用代码时，这类操作能保证代码的安全与整洁。

* **`Shift + F6`**：安全重命名。用于修改类名、方法名或变量名。IDEA会自动更新项目中所有引用了该名称的地方，避免手动修改带来的遗漏。
* **`Ctrl + Alt + L`**：格式化代码。按照设定的代码规范自动调整缩进、空格和换行。
* **`Ctrl + Alt + O`**：优化导入。自动删除类中未使用到的`import`语句，保持代码头部整洁。
* **`Ctrl + D`**：复制当前行或选中的代码块到下一行。
* **`Ctrl + Y`**：删除当前行。

## 四、 运行与调试
对于逻辑验证和排错，掌握快捷键可以让你专注在代码逻辑上。

* **`Shift + F10`**：运行当前项目或上一次运行的配置。
* **`Shift + F9`**：以Debug模式运行。
* **`F8`**：在Debug模式下，单步步过 (Step Over)，不进入方法内部。
* **`F7`**：在Debug模式下，单步步入 (Step Into)，进入自定义方法内部查看逻辑。
* **`F9`**：在Debug模式下，恢复程序运行 (Resume Program)，直到遇到下一个断点。
